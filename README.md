# Task Manager
A Simple Task Manager made in C++ Works in the Terminal

### Description
A Simple Task Manager made in C++ works in the terminal lets you add and show all tasks.

### Features
This Project Uses the Plain text `.txt` file to manage and get tasks and when created the script would name it `Tasks.txt` you don't have to create it as the script creates it manually

The script lets you add a task and show all the tasks

### Functions
| Function | Description |
| -------- | ----------- |
|   Add    |      Adds a new task to the tasks list       |
|   Get    |      Gets the tasks list from the file and print it out       |

### How to Run
to run the script just basically do:
```
./main
```
> Note: I always build the file when I add a new feature so you don't have to
